Probability-Guided Program Reduction(Research project)

2020.02
  • Given a program P that exhibits a certain property, the goal of program reduction is to minimize P to a smaller variant which still exhibits the same property.
  • This technique can be used in compiler testing, software debloating, etc.
  • We use a novel method which first splits the program into elements under certain granularity and then attaches probability to each element representing their possibility to be reduced. Taking relationship between elements into consideration, we adjust elements' probability according to information we get during the reduction process to achieve smaller reduction time as well as output size.
  • We get better results in terms of speed(reduction time,34.35% accelerated) and quality(size of reduced programs,19.09% smaller) on two recognized benchmarks compared to state-of-art method.
